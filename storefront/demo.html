ğŸ”Œ API Integration

The app fetches data from these Laravel API endpoints:

GET /api/storefront/{tenant}/theme - Theme settings and configuration GET /api/storefront/{tenant}/page/{slug} - Page content with sections GET /api/storefront/{tenant}/products - Product list GET /api/storefront/{tenant}/products/{slug} - Individual product

ğŸ¨ Theme System

The theme system is fully extensible with a registry pattern:

Section Types Available:

  • hero - Hero banner with title, subtitle, CTA
  • product_grid - Product listing with filters and pagination
  • featured_products - Featured products showcase
  • content - Rich text content with optional image

ğŸš€ Getting Started

  1. Ensure your Laravel API is running with the multi-tenant storefront endpoints
