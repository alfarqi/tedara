🏪 Storefront React App Demo

📋 Project Overview

This is a multi-tenant storefront application built with React, TypeScript, and Vite. The app automatically detects the tenant from the URL path and fetches content from the Laravel API.

✅ Features Implemented:
  • Multi-tenant architecture with automatic tenant detection
  • React Router with tenant-aware routing
  • Theme system with classic theme and extensible registry
  • API integration hooks (useTheme, usePage)
  • Responsive design with Tailwind CSS
  • TypeScript for full type safety

🌐 URL Structure

The app automatically detects the tenant from the first path segment:

https://example.com/salmeen-electronics/ → Tenant: salmeen-electronics https://example.com/fatooma-store/catalog → Tenant: fatooma-store https://example.com/demo-store/product/sample-product → Tenant: demo-store

🛣️ Routes

  • / - Home page (fetches page with slug 'home')
  • /catalog - Product catalog page
  • /product/:slug - Individual product page
  • /:slug - Dynamic page (about, contact, etc.)

🔌 API Integration

The app fetches data from these Laravel API endpoints:

GET /api/storefront/{tenant}/theme - Theme settings and configuration GET /api/storefront/{tenant}/page/{slug} - Page content with sections GET /api/storefront/{tenant}/products - Product list GET /api/storefront/{tenant}/products/{slug} - Individual product

🎨 Theme System

The theme system is fully extensible with a registry pattern:

// Theme Registry export const themeRegistry = { classic: { Layout: ClassicLayout, sections: { hero: ClassicHero, product_grid: ClassicProductGrid, content: ClassicRichText, }, }, };

Section Types Available:

  • hero - Hero banner with title, subtitle, CTA
  • product_grid - Product listing with filters and pagination
  • featured_products - Featured products showcase
  • content - Rich text content with optional image

🚀 Getting Started

  1. Ensure your Laravel API is running with the multi-tenant storefront endpoints
  2. Run npm run dev to start the development server
  3. Visit http://localhost:5173/demo-store/ to see the demo tenant
  4. The app will automatically fetch theme and page data from the API

Note: The app includes sample data fallbacks when the API is not available, so it will work even without the Laravel backend.

📱 Responsive Design

The app is fully responsive and works on:

  • Desktop computers
  • Tablets
  • Mobile phones

Built with Tailwind CSS for consistent, mobile-first design.

🔧 Customization

To add new themes or sections:

  1. Create new theme folder: src/theme/modern/
  2. Implement Layout and section components
  3. Register in theme/registry.ts
  4. The theme will be automatically available
